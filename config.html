Ollama Local — instalação manual
O app não instala o Ollama pra você. Siga o passo-a-passo abaixo (uma vez só) e depois é só usar.
1. Instalar o Ollama (uma vez):
curl -fsSL https://ollama.com/install.sh | sh
Ou baixe direto em ollama.com/download (tem versão Windows/macOS/Linux).
2. Baixar o modelo escolhido acima (cada um faz só uma vez):
ollama pull qwen2.5-coder:7b
3. Verificar se está rodando:
ollama serve     # opcional — o instalador já deixa rodando como serviço
Requisitos mínimos por modelo
Modelo RAM VRAM Tamanho
qwen2.5-coder:7b8 GB+6 GB+~4.7 GB
llama3.1:8b8 GB+6 GB+~4.9 GB
gemma3:12b16 GB+10 GB+~8.1 GB
Sem GPU? Roda mesmo assim na CPU/RAM — só fica mais lento (5-15 tok/s). Recomendado: 32 GB RAM DDR4.
Ferramentas avançadas e Acesso a diretórios são desativados ao usar Ollama Local nesta versão (mantém simples e estável). Use ChatGPT se precisar dessas features.
Gemini CLI — instalação manual
O app não instala o Gemini CLI. Configure antes de usar:
1. Instalar:
npm install -g @google/gemini-cli
2. Autenticar:
gemini auth login
3. Verificar:
O Gemini CLI gerencia arquivos, terminal e ferramentas diretamente. O Helper Node apenas envia prompts e exibe as respostas.
Ferramentas avançadas e Acesso a diretórios são desativados neste modo — o CLI gerencia tudo.
Claude Code CLI — instalação manual
O app não instala o Claude Code. Configure antes de usar:
1. Instalar:
npm install -g @anthropic-ai/claude-code
2. Autenticar (abre browser):
claude
3. Verificar:
O Claude Code CLI gerencia leitura, edição, bash e ferramentas de forma nativa. O Helper Node envia prompts e exibe os resultados em tempo real, incluindo diffs dos arquivos editados.
Ferramentas avançadas desativadas neste modo — o CLI gerencia tudo internamente.
